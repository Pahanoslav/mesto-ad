# Mesto

Интерактивная лента карточек с местами. Финальная версия проекта: данные пользователя и карточки
загружаются и сохраняются на сервере через REST API, формы валидируются на лету, а по клику на
кнопку «i» в карточке открывается модальное окно с подробной информацией о ней (описание, дата
создания, владелец, количество лайков и список лайкнувших пользователей).

## Демо

[Открыть проект на GitHub Pages](https://pahanoslav.github.io/mesto-ad/)

## Деплой

Сборка автоматически публикуется на GitHub Pages через GitHub Actions
(`.github/workflows/deploy.yml`) при каждом пуше в ветку `main`. Workflow собирает
проект (`npm run build`) и публикует содержимое `dist` в ветку `gh-pages` этого
репозитория, где включён GitHub Pages.

## Команды

Установка зависимостей:

```bash
npm install
```

Запуск локального сервера разработки:

```bash
npm run dev
```

Сборка проекта:

```bash
npm run build
```

Публикация собранного проекта на GitHub Pages:

```bash
npm run deploy
```
